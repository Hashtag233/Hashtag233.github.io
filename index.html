发表于 2021-01-25 | 分类于 Learning |

通知原文

2021年江苏省中学生“英才计划”选拔工作已于近日完成，共有198名学生参与申报，其中142名学生入围面试。根据学生学科潜质测试及面试成绩，共选拔出62名优秀中学生进入本年度培养计划。

省级管理办公室将学员名单报送全国管理办公室批准后，将举行拜师仪式，时间地点另行通知。

什么是Markdown

  • Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编辑文档。

  • Markdown语言在2004年由约翰·格鲁伯（英语：John Gruber）创建。

  • Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

  • Markdown 编写的文档后缀为 .md, .markdown。
