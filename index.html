什么是Markdown

  • Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编辑文档。

  • Markdown语言在2004年由约翰·格鲁伯（英语：John Gruber）创建。

  • Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

  • Markdown 编写的文档后缀为 .md, .markdown。
